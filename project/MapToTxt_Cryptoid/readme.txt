Place all images to be converted to terr files in "/maps"
All converted map images will be placed in "/terr"

type all file name beginnings in "fileLeads.txt", each with its own line
type the file extension of the image maps in "fileType.txt", only the first line will be read
type the color map into "colorMap.txt", with each color having its own line in the following format : "#xxxxxx z" or "#aaxxxxx z", where 'z' is the character to be written out, and 'a' is alpha
type the desired destination of the .terr files into 'fileDestination.txt' on the first line

errors generated by missing txt files listed above or mismatched pixel colors will be written to "error.txt"
